3D Characterizer

3C is a web-based application that is designed for creating a 3d model of humans.User can upload 360 video that has his/her desired character.
The user can wait till the 3d model is processed. The system will give model file. The model can be imported in Blender or MeshLab then to
Unreal Engine.
